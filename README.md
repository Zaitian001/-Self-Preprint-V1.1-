# 个人完全自主“自预印本 Self-Preprint”安全发布系统 (V1.1)

> **知识的自我主权：学术优先权不再由出版商的印章决定，而由物理货币的不可伪造性与密码学时空锚共同担保。**

本系统严格依据发明专利 **CN101320459A《通货信用保证系统防伪方法》**（发明人：伍宝林 Baoliin Zaitian Wu, ORCiD: 0009-0003-2051-5247）的技术逻辑设计。它为全球独立科学研究者提供了一套完全自主、即时可验、不可篡改、防劫持的科学知识优先权发布规范。

---

## 🌌 一、 系统概述

在传统学术体系中，科学发现的优先权依赖于第三方中心化机构（如期刊、学术会议、第三方预印本平台）的审核与时间戳戳记。这带来了时间滞后、学术审查、甚至成果被劫持的风险。

**自预印本（Self-Preprint）** 颠覆了这一范式：
* **发布者既是卖方也是买方**：作为知识的创造者（卖方）向全人类提供科学发现（货品）；作为知识生态的参与者（买方），所有阅读者均可即时验证该知识的真实信用状况。
* **国家信用背书**：将论文与国家公开发行、合法流转的法定纸币序列号进行双向绑定，利用国家级物理防伪技术为个人数字学术产权提供强力的物理硬锚。

## 🛡️ 二、 安全三维防御原理

为了防止恶意剽窃者对自预印本论文进行劫持、伪造和时间篡改，本系统（V1.1）部署了三道安全防御盾牌。


### 1. 物理盾牌：物理储备池预公开（Proof of Reserve）
* **原理**：发布者在系统运行之初，向银行依法获取批量连号物理货币（如 100 张连号人民币），并将全套图像预先公开上传至本仓库。
* **防御效果**：任何企图挪用本系统纸币图像伪造论文的攻击者，其发布时间都会晚于仓库中储备池图片的最早提交时间，在 Git 历史记录面前瞬间暴露。

### 2. 密码盾牌：像素级双向哈希缠绕（Bi-directional Splicing）
* **原理**：发布论文时，计算论文内容的 `SHA-256` 唯一指纹，并使用图像水印技术将该哈希字样以半透明、像素级不可逆的方式覆写于绑定纸币的图像上。
* **防御效果**：攻击者无法将你的论文内容剥离后去绑定他们自己的纸币，因为你公开的纸币图像中已经永久带有你特定论文的哈希像素印记；他们也无法通过 AI 伪造纸币图像，因为图像边缘和微缩防伪特征一旦被篡改，肉眼和机器比对会立刻识破。

### 3. 时空盾牌：分布式矩阵时间锚（Matrix Time-Anchor）
* **原理**：在论文发布当天，将成果同步分发至全球 7 个主流互联网巨头平台（如微信、知乎、微博、X 等）。以第一时间（首发节点 $T_1$）到第七时间（终点节点 $T_7$）构建一个物理时空的“多维脚印”。
* **防御效果**：攻击者要想伪造比你更早的发布时间，必须同时入侵并修改这 7 家全球顶级科技巨头各自独立运行的时间数据库。这在现实物理世界中是不可能的。

## 🚀 三、 创作者使用指南

### 步骤 1：构建信用储备池
1. 从银行提取一批（建议 100 张）同面额连号纸币。
2. 将它们按顺序存入本仓库的 `/CURRENCY_REGISTRY/` 目录下。
3. 拍摄高清大图（需清晰展现序列号与水印安全线等物理特征）。
4. 在本仓库的 `CURRENCY_REGISTRY.md` 中注册该区间（例如：`CN-HB00000001` 至 `CN-HB00000100`），并一次性提交（Commit）推送至 GitHub，锁定储备池起始时间。

### 步骤 2：撰写并制备论文
1. 完成论文写作（建议使用 Markdown 格式，保存至 `/PREPRINTS/` 目录）。
2. 从储备池中取出当前待用的一张物理纸币，计算该论文不含身份声明区块的 **SHA-256 值**。
3. 将该 SHA-256 值通过脚本或图像处理工具，作为物理水印添加在该纸币图像的指定空白区域。
4. 在论文扉页或摘要后嵌入以下标准《自预印本身份声明》：

## 🔍 四、 读者验证手册

任何人都可以通过以下三个步骤，在 3 分钟内自主、独立地验证任何一篇自预印本论文的真实性与时间优先权：

1. **第一步：指纹核对（1 分钟）**
* 下载论文文件，计算其排除声明区后的 `SHA-256` 值。
* 比对该值是否与《自预印本身份声明》中的 **“本文内容唯一哈希”** 完全一致。


2. **第二步：实物担保验证（1 分钟）**
* 访问本 GitHub 仓库的 `/CURRENCY_REGISTRY/` 文件夹。
* 调取该论文绑定序列号（如 `CN-HB00000001`）的纸币图像。
* 核对纸币图像上由像素级覆写的哈希值是否与你在第一步中算出的哈希值完美契合，并肉眼比对纸币防伪水印。


3. **第三步：时间锚验证（1 分钟）**
* 随机抽取论文末尾“矩阵发布时空链”中的 2 个以上的平台 URL。
* 点击进入第三方平台的官方页面，核实平台的系统发布时间是否在 $[T_1, T_7]$ 区间内。
* 若所有要素完全匹配，则该论文的学术优先权、发布时间、完整性在法理与技术上被确立。
